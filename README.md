# API de Ordens de Serviço com Spring Boot e RabbitMQ

API REST para receber pedidos de ordens de serviço, utilizando Spring Boot e integração com um sistema de fila RabbitMQ para processamento assíncrono e banco de dados NoSQL MongoDB para armazenamento.

## Tecnologias Utilizadas

- Java 21
- Spring Boot
- MongoDB
- RabbitMQ
- Docker
